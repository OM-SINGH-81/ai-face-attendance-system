# AI-Based Face Recognition Attendance System

## Features
- Real-time face recognition
- Anti-proxy using eye-blink liveness detection
- Automatic check-in / check-out
- Multi-camera support
- Django authentication

## Tech Stack
- Python, Django
- OpenCV, FaceNet, MTCNN
- SQLite
- HTML, CSS, Bootstrap

## Use Case
Automated attendance for schools, colleges, and institutions.
